## Crowdfunding Campaign Analysis

## Objective
Analyze crowdfunding campaign data to identify key factors that influence project success, including funding goals, category performance, and launch timing.

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Data Visualization  
- Statistical Analysis  

## Dataset
- 1,000 crowdfunding projects  
- Includes data on funding goals, outcomes, backers, categories, and timelines  

## Key Analysis Performed
- Created calculated fields:
  - Percent Funded  
  - Average Donation per Backer  
- Cleaned and transformed raw data:
  - Split category fields into parent/sub-categories  
  - Converted Unix timestamps into readable dates  
- Built pivot tables to analyze:
  - Campaign outcomes by category and sub-category  
  - Success rates based on funding goals  
  - Trends based on launch date  
- Developed charts and dashboards to visualize:
  - Success vs failure rates  
  - Goal vs outcome relationships  
  - Time-based trends  

## Key Insights
- Campaigns with **lower funding goals** had significantly higher success rates  
- Certain categories consistently outperformed others  
- Launch timing showed trends in campaign success (seasonality impact)  
- Successful campaigns had **higher average backers and funding efficiency**
