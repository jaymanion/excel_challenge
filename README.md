# Crowdfunding Campaign Analysis

A data-driven analysis of crowdfunding campaigns, leveraging Excel-based dashboards and exploratory data analysis to uncover patterns in campaign performance, funding success, and user behavior.

---

## Objective
Analyze crowdfunding campaign data to uncover key drivers of project success, including funding goals, category performance, and launch timing. This project aims to support data-driven decision-making through structured analysis and visualization.

---

## Tools & Technologies
- Microsoft Excel (Advanced)  
- Pivot Tables & Calculated Fields  
- Data Cleaning & Transformation  
- Exploratory Data Analysis (EDA)  
- Data Visualization & Dashboarding  

---

## Dataset
- 1,000 crowdfunding projects  
- Includes data on:
  - Funding goals  
  - Campaign outcomes  
  - Number of backers  
  - Categories and sub-categories  
  - Campaign timelines  

---

## Key Analysis Performed
- Engineered key performance metrics, including:
  - Percent Funded  
  - Average Donation per Backer  
- Cleaned and transformed raw data:
  - Split category fields into parent and sub-categories  
  - Converted Unix timestamps into readable date formats  
- Conducted multi-dimensional analysis using pivot tables to evaluate:
  - Campaign outcomes by category and sub-category  
  - Success rates across different funding goal ranges  
  - Trends based on campaign launch timing  
- Developed dashboards and visualizations to highlight:
  - Success vs. failure rates  
  - Relationships between funding goals and outcomes  
  - Time-based and seasonal trends  

---

## Key Insights
- Campaigns with **lower funding goals** demonstrated significantly higher success rates  
- Certain categories consistently outperformed others in funding success  
- Campaign launch timing revealed **seasonal trends** impacting performance  
- Successful campaigns showed **higher backer engagement and funding efficiency**
